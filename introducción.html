Examen Programacion r

Capitulo 2 Introducción

En este capítulo se analiza el conjunto de datos de atenciones de urgencias del año 2023 del pais Chile. En la revision y procesamiento de los datos, se requiere el uso de algunas librerias del lenguaje r, las cuales son instaladas y cargadas para el uso en el proyecto. Además se explica la forma de visualizar el proyecto, por las via de compilacion del proyecto y por via publicación en formato html, utilizando la plataforma Github

2.1 Librerias

  • readr: Libreria para la lectura del conjunto de datos
  • dplyr : Libreria para la manipulación de datos
  • ggplot2: Libreria para mostrar graficamente los datos
  • factoextra: Libreria para gráficar el clustering

2.2 Visualizacion

La forma de visualizar el proyecto es compilando el proyecto o visualizando a traves de la plataforma de github

2.2.1 Compilando el proyecto

Debe descargar el proyecto comprimido, que se encuentra en desde github Proyecto ExamenR Despues de descargado el proyecto, debe descomprimiry en la consola de Rstudio, ejecutar el sigunente código para compilar el proyecto

  • bookdown::render_book(“index.Rmd”, “bookdown::gitbook”)

Una vez compilado debe ir al directorio _book , abrir el archivo index.html
